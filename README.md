# COMO EXECUTAR

## windows
 ```python
   python -m venv venv 
   .\venv\Scripts\activate
   pip install -r requirements.txt
   python main.py
 ```

## Mac ou Linux
 ```python
   python3 -m venv venv
   source venv/bin/activate
   pip3 install -r requirements.txt
   python3 main.py
 ```
